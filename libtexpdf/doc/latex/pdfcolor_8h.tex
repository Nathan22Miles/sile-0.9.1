\subsection{Detailed Description}
P\+D\+F Color handling functions. 

{\ttfamily libtexpdf} provides functionality for setting various colors within the generated P\+D\+F. Most typically when generating P\+D\+Fs, you will want to create a color object using one of the initializing functions like so\+: \begin{DoxyVerb}pdf_color white, black, grey;
texpdf_color_rgbcolor(&white, 1.0, 1.0, 1.0);
texpdf_color_rgbcolor(&black, 0,   0,   0);
texpdf_color_rgbcolor(&grey,  0.5, 0.5, 0.5)
\end{DoxyVerb}


You will then probably want to either set the drawing pens for stroke and fill to your chosen color permanently\+: \begin{DoxyVerb}texpdf_color_set(pdf, black, grey); // Black stroke, grey fill
\end{DoxyVerb}


Or temporarily\+: \begin{DoxyVerb}texpdf_color_push(pdf, black, grey);
... Do some work here ...
texpdf_color_pop(pdf);\end{DoxyVerb}
